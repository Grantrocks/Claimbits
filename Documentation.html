    Script Installation
    1. Upload all files from the folder ClaimBits to your site, in the root folder.
    2. Create the database (use cpanel or whatever database management client you have available)
    3. Change file system/database.php permissions to 777
    4. Go to http://yourdomain.com/install and follow the steps to install the script
    5. Change file system/database.php permissions to 644
    6. Create a Cron Job (make it run every minute) and call that file: /system/cron/cron.php

      Example: * * * * *     php /home/[YOUR_USERNAME]/public_html/system/cron/cron.php

    Installation complete, delete folders install and you're done.


    To manage site settings, users, websites, etc, go to http://yourdomain.com/admin-panel and login with your admin account.


    Script Update

    When a new version is released, you get notified into your admin panel. To get latest version of the script, all you have to do is to download script again from MN-Shop.com -> Customer Area -> Downloads section, then follow the instructions listed bellow.
